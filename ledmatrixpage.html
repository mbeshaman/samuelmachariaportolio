LED MATRIX PROJECTS

led matrix display

the 8x8 matrix display is based on MAX7221 decoder IC. The LEDs can be of different colours. Several matrices can be daisy chained to create a bigger display. The message displayed on the led matrix can be updated wirelessly by means of a smartphone application.
