ESP NOW PROJECT

ESP NOW is a connectionless communication protocol that is used for sharing small data packets between two ESP boards developed by ESPRESSIF.It offers a persistent connection between ESP nodes,this means once two ESP nodes are connected with each other (wirelessly) and suddenly one of the ESP devices loses poweror restarted,the device will again (automatically) establish a connection with the other node to which it was connected before the power-reset.

serial messages from both nodes

The above image shows messages from a node connected to arduino(/dev/ttyACM0) and another node conneced to FTDI usb to serial converter(/dev/ttyUSB0).

One node sends its MAC address to the other node which responds by sending its own MAC address back to the first node.
